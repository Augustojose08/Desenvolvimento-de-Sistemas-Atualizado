Front-End

O que é Front-End?

O Front-End é a parte de um site ou aplicativo que interage diretamente com o usuário. Ele envolve a criação e o design de páginas web, utilizando linguagens como HTML, CSS e JavaScript.

Conteúdo do Curso

O curso de Front-End abrange os seguintes tópicos:

  • HTML: Estruturação do conteúdo web
  • CSS: Estilização e design de páginas
  • JavaScript: Interatividade e comportamento dinâmico
  • Frameworks: React, Vue.js, Angular
  • Boas práticas de acessibilidade e SEO
  • Ferramentas de versionamento: Git e GitHub
  • Responsive Web Design (Design Responsivo)

Objetivo do Curso

Capacitar o aluno a desenvolver interfaces de usuário atrativas, acessíveis e funcionais, criando páginas que se adaptam a diferentes dispositivos e tamanhos de tela, além de integrar boas práticas de usabilidade.

Público-Alvo

Esse curso é ideal para iniciantes que desejam aprender a criar sites do zero, ou para desenvolvedores que querem aprimorar suas habilidades no desenvolvimento Front-End.
